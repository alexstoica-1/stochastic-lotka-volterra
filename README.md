# Stochastic Lotka-Volterra Simulation

O sa va dau invite ca si collaboratori la proiectul asta.

Dupa ce acceptati trebuie sa faceti in felul urmator:
1. pe pagina web de la repository apasati pe butonul verde cu code si apoi dati copy link
2. Dupa ce deschideti vscode, scrieti in terminal in felul urmator:

```
git clone LINKULVOSTRU
git status 
git pull
```

De aici putem sa facem branches separate cu numele noastre daca vreti ca sa dam update separat la proiect, sau putem sa dam push la tot ce e nou direct pe main branch, doar sa ne anunatam inainte cand schimbam ceva.

Mereu cand vreti sa lucrati dati:

```
git pull
```

ca sa fiti siguri ca sunteti up to date.

Si cand vreti sa trimiteti ce ati facut:

```
git add .
git commit -m "ce mesaj vreti voi"
git push origin BRANCHUL_VOSTRU
```
